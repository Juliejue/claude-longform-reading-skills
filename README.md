# Claude Longform Reading Skills

Three portable Claude/Agent Skills for deep longform reading:

- `ai-longform-reading-coach`: AI frontier technology and humanities deep-reading coach.
- `art-culture-reading-coach`: art, culture, aesthetics, and philosophy deep-reading coach.
- `crypto-longform-reading-coach`: Web3 sociology and mechanism-design deep-reading coach.

These skills are designed for Chinese deep reading. They emphasize substitute reading, detailed logic reconstruction, bilingual highlights, active batching for long articles, and concrete action toolkits.

## Usage

Copy any skill folder into a compatible skills directory, keeping the folder name and `SKILL.md` together.

Each skill is independent and can be installed separately.
